# Play & Vibe🎧  
# About The Project  
🎤🎤"This is a simple and user-friendly music player built with HTML and CSS, designed with a clean look and smooth experience."  
# ✨ Features  
🎧play songs using audio player  
▶️play and pause songs  
⏭️skip to next track  
⏮️Go back to previous track  
🔊Volume control  
📂Multiple song list  
🎨Styled using css  
🦚Fast and lightweight
